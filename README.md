# Pudding 🍮
Multiplexed Information and Computing Agent (多路复用信息与计算代理)

Pudding 是一款功能强大的独立 GUI 客户端，致力于让每位用户都能轻松使用 AI 编码代理（AI Coding Agents）。基于 Agent Client Protocol (ACP) 协议，它提供了一个简洁直观的聊天界面，让您可以摆脱特定 IDE 或终端操作的束缚，与各种编码代理高效协作。

## ✨ 核心特性
🔌 ACP 原生支持: 无缝连接并切换任何兼容 ACP 协议的代理，如 opencode、Claude Code、OpenClaw等。
🎙️ 语音交互: 增强的语音驱动编码功能，提供更自然的交互体验。
📁 编辑器无关: 独立的桌面运行环境，不绑定任何特定代码编辑器或 IDE。
🧠 会话持久化: 完整的对话历史管理，支持跨本地工作区暂停及恢复进度。
🚀 项目导向: 按项目目录组织代理交互，保持上下文的一致性与高效性。
🎨 优质 UI/UX: 现代且响应迅速的界面，支持语法高亮、Markdown 渲染及实时流式更新。
🚀 快速入门
## 前置条件
Pudding 需要与本地编码代理进行交互（内置了 opencode）。

## 安装步骤
从 Releases 页面下载适合您平台的最新版本：
Apple Silicon (M1/M2/M3): Pudding-{version}-arm64.dmg
Intel Mac: Pudding-{version}-x64.dmg
将 Pudding 拖入您的 Applications (应用程序) 文件夹。
打开 Pudding，并在 Settings (设置) 中配置您喜欢的代理。
## 快速开始
选择项目: 打开您希望代理工作的目录。
选择代理: 确保在侧边栏或设置中选择了目标代理。
开始聊天: 输入您的指令（例如：“分析此项目的结构”），即可开始体验！
## 🛠️ 技术栈
框架: Electron & React
协议: Agent Client Protocol (ACP)
样式: Tailwind CSS & Framer Motion
数据库: 使用 SQL.js 管理本地会话
